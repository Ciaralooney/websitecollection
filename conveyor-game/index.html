Conveyor Panic is an arcade game where you sort items off a conveyor belt into bins before time runs out. The goal is to sort as many items as you can within 60 seconds. It's a 2D web game built in Unity.

Design

The core loop is built around quick decision making rather than precision. Items move along the conveyor belt and the player only has a short window to recognise each one and sort it into the correct bin before it reaches the end of the belt. Capping the round at 60 seconds was a deliberate choice as I wanted it to be a quick game but the player has long enough in a round to figure out mistakes and improve.
